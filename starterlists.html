Starter deck lists, by set and region

The following is a list of all the fixed (non-random) cards that appeared in each of the Magi-Nation starter decks. The first printing of the base set starter decks contained no random cards, but starting with the Unlimited printing, each starter deck contained some number of fixed and random cards.
